# 🚀 Telegram Control Tool (Session Based)

A simple Telegram group management tool using Telethon.

## ⚙️ Features
- Login with API ID & API HASH
- Session based login (no OTP every time)
- Promote/Demote admin
- Ban/Unban users
- View account info
